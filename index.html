Khái niệm

Lệnh git log trong Git cho phép bạn xem lịch sử commit của một kho lưu trữ, hiển thị danh sách các commit đã được thực hiện.
Đây là một công cụ hữu ích để kiểm tra và theo dõi các thay đổi trong lịch sử dự án của bạn.

Các tùy chọn thông dụng của git log

- git log: Hiển thị danh sách các commit, mỗi commit bao gồm SHA-1 hash, tác giả, ngày tháng, và thông điệp commit.
- git log --oneline: Hiển thị mỗi commit trên một dòng duy nhất(id+message).
- git log -n : Hiển thị n commit gần nhất. Ví dụ, git log -n 3 sẽ hiển thị 3 commit gần nhất.
- git log --graph: Hiển thị lịch sử commit dưới dạng biểu đồ cây.
- git log --stat: Hiển thị các tệp đã thay đổi và thống kê các thay đổi cho mỗi commit.
- git log --pretty=format:"%h - %an, %ar : %s": Tùy chỉnh định dạng hiển thị của log.
